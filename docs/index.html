Este é um post criado e publicado despois de ter subido e despregado o sítio web HUGO no Github. É umha operaçom rutinária Cria-se co comando “hugo new DIRECTORIO/NOME_POST.md” Cada vez que se quere actualizar o contido é necessário criar os novos ficheiros de produçom (dentro do directório /docs/). O nome “docs” vém dado por exigências de Github Pages, pois este é o único nome que admite para o despregue da página.
